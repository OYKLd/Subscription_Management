# Subscription_Management

## Description
Application web permettant de gérer ses abonnements (Netflix, Spotify, etc.) en suivant les coûts, les dates de paiement et le total des dépenses.  
Ce projet est le **trente-quatrième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Implémenter un CRUD complet
- Manipuler des données financières
- Gérer les dates de paiement
- Calculer des totaux dynamiques
- Concevoir un mini dashboard clair

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage

---

## Fonctionnalités
- Ajout d’un abonnement (nom, prix, fréquence, date)
- Liste des abonnements
- Calcul du total mensuel
- Affichage du prochain paiement
- Suppression d’un abonnement
- Sauvegarde locale des données

---

## Design & UX
- Dashboard avec total mensuel en évidence
- Cartes pour chaque abonnement
- Informations claires (prix, date, fréquence)
- Interface propre et structurée
- Responsive (mobile et desktop)
